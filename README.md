## Features

- Static HTML/CSS layout practice.
- Design source files for multiple page concepts.
- JavaScript modules for navigation, hero, and on-scroll interactions.
- Local JSON data files for layout/content experiments.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- JSON

## Quick Start

Clone the repo and open `index.html` in your browser, or use the VS Code Live Server extension.

```bash
git clone git@github.com:angelajholden/practice-layouts.git
```

## Project Structure

- `index.html` - main page.
- `modules/` - JavaScript behavior modules.
- `data/` - local data files.
- `psd/` - design sources.
- `images/` and `svg/` - visual assets.
